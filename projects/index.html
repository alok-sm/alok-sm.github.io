Projects

Studying the Wisdom Of Crowds at Scale

The Wisdom of Crowds is a phenomenon where the aggregation of decisions/answers in groups are often better than of any single member of the group. Worked with a team of researchs from Stanford University lef by Dr. Sharad Goel in order to build the Web platform for the Experiment on Amazon Mechanical Turk and in the analysis of the experiment data.

Hosted at wisdomofcrowds.stanford.edu

ACM Publication

Maching Learning Approach to Auto-scaling CloudFoundry

CloudFoundry, an open source PAAS cloud software utilized a reactive approach to scaling up user applications. Our project was the investigation and Application of machine learning techniques such as Neural Networks and Decision trees to preempt server load. By doing this, we can scale up or scale down CloudFoundry Application based in order to minimize bad QoS.

Time series sub-sequence search on GPU

Implemented and Optimized UCR’s Flavor of the Dynamic Time Warping algorithm for Time Series Sub-sequence search on Nvidia CUDA. Achieved 590% versus the original algorithm on a NVidia GT-750Ti.

Excavator - A Platform for automated Web Data Mining

Built a platform for automating mining and storing data from Web Pages as a a part of the Software Engineering course in the 6th semester. Users of this platform can schedule data scraping tasks to run periodically to scrape data from a web page. This scraped data was accessible to the users through a REST API. Github Link to the Excavator project

Augmented reality with image recognition on Android

Built an Augmented Reality experience on the Android which works on color descriptor based image recognition to recognizes images taken by the Android camera and matches it with a database on a remote server to serves up relevant video content on top of the android camera feed.

Website development for TEQIP

Created a student and faculty facing website with Ruby on Rails for providing details regarding the activities conducted by TEQIP’s PESIT chapter.
