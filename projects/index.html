Projects

Studying the Wisdom Of Crowds at Scale

The Wisdom of Crowds is a phenomenon where the aggregation of decisions/answers in groups are often better than of any single member of the group. Worked with Prof. Sharad Goel of Stanford in order to build the Web platform for the Experiment, hosting the experiment on Amazon Mechanical Turk and in the analysis of the experiment data.

Hosted at wisdomofcrowds.stanford.edu

Utilizing Open data sources to enhance Bing SERP triggering

During my summer internship in the summer of 2015 at the Microsoft Indian Development Center - Hyderabad, I worked on utilizing Open data sources such as Wikipedia, Wikidata and DBpedia to enhance the triggering of the Bing Search Engine Results Page (see image)

My work was able to enhance the triggering of the SERP for around 300,000 entities.

Perceptron Approach to Auto-scaling CloudFoundry

CloudFoundry, an open source PAAS cloud software utilized a reactive approach to scaling up user applications. We built a Perceptron Learning algorithm based auto-scaling system for preemptive scaling of user applications in order to minimize bad QoS time.

Time series sub-sequence search on GPU

Implemented and Optimized UCR’s Flavor of the Dynamic Time Warping algorithm for Time Series Sub-sequence search on Nvidia CUDA. Achieved 12x versus the original algorithm on a mid range Nvidia GPU

Excavator - A Platform for automated Web Data Mining

Built a platform for automating mining and storing large volumes of data from Web Pages as a mini project for the Software Engineering course in the 6th semester. Users of this platform can schedule data scraping tasks to run periodically to scrape data from a web page. This scraped data was accessible to the users through a REST API. Github Link to the Excavator project

Augmented reality with image recognition on Android

Built an Augmented Reality experience on the Android which works on color descriptor based image recognition to recognizes images taken by the Android camera and matches it with a database on a remote server to serves up relevant video content on top of the android camera feed.

Website development for TEQIP
